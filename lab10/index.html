What is XML?

XML stands for Extensible Markup Language. It is a language used to describe data in a flexible way. Meaning you can represent structures of information in human readable text. XML is a public standard developed by W3C (The World Wide Web Consortium).

How is XML used?

XML can be used to store and arrange data, as well as exchange information between organizations and systems. XML can essentially work out of sight to simplify the process of creating HTML docs for big web sites.

What does XML look like?

XML code looks human readable. A markup language is information added to a document that enhances its meaning in certain ways. XML accomplishes this with descriptive markup symbols/tags. What is JavaScript?

JavaScript is the final piece of the puzzle that brings websites together and really makes them come alive. It's what you need to add interactivity and fun to a website! JavaScript is a client side programming language that controls multimedia within webpages. JavaScript has nothing to do with Java the language.

How is JavaScript related to HTML and CSS?

Like mentioned above, JavaScript is used to add interactivity to a website. JavaScript is related to HTML and CSS in that it is a core component of building webpages and adding more functionality to them.

What are some common JavaScript elements, properties, methods, and functions?

Some common types of JavaScript elements, properties, methods, and functions include:

  • Variables - elements which allow you to store data - be it strings of characters, numbers, or other data types. The example below stores the name Toyota in a variable named car for future use:

    let car = "Toyota";

  • Loops - elements which allow you to repeat a block of code. The example below increments through a range of 0-5 using the variable i as a placeholder:

    for (let i = 0; i < 5; i++) {
        console.log(i);
    }

  • .href - this is a common property used to return the href attribute of an element. See the example below:

    let link = document.getElementById("Link to website");
    link.href = "https://www.link.com";

  • length - a property commonly used to return the number of characters or elements within a string or array. In the example below we return the number of characters in the string named "Name":

    let name = "Jordan";
    console.log(name.length);

  • toString() - a commonly used JavaScript function which turns the elements inside an array into a string. In the example below we convert the array cars into a string instead:

    let cars = ["Mazda", "Honda", "Prius"];
    let string = cars.toString();
    console.log(string);

  • indexOf() - a commonly used function to return the position of a specified element within an array. In the example below: indexOf returns the index of the 2 element in the array named numbers:

    let numbers = [1, 2, 3, 4, 5];
    let index = numbers.indexOf(2);
